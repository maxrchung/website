POKEMANS

Last edited 5 Apr. 2015

Info

POKEMANS is basically Pokemon with naked people. It's a TCP networked version of Pokemon's Battle Frontier/Battle Tower/Battle Factory where players, er, sorry, rich men, draft 3 naked men and fight other rich men with them. Or something like that. Also, I use the term "TCP networked" very loosely because I'm pretty sure I try catched pretty much all my network code to ignore errors ¯\_(ツ)_/¯. Anyways, as of now, the game isn't yet complete. I think we want to polish it up and then release it a bit later or never.
